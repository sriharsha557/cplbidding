# SQL Scripts

This folder contains all SQL scripts for database management.

## Database Schema
- `supabase-schema.sql` - Main database schema definition

## Data Insertion
- `cpl_auction_2025_insert.sql` - Generated SQL for CPL 2025 players
- `insert_cpl_2025_data.sql` - Master insertion script with instructions
- `insert-real-data-template.sql` - Template for custom data insertion

## Data Management
- `supabase-cleanup-scripts.sql` - Complete data management toolkit
- `prepare-for-real-data.sql` - Quick cleanup for production data

## Usage
1. Review the script you need
2. Open Supabase SQL Editor
3. Copy and paste the SQL
4. Run the script
5. Verify with included SELECT queries

**Note**: Always backup your data before running DELETE or UPDATE statements!
